# COVID-19_Seg
[![](https://img.shields.io/badge/python-3.6%2B-green.svg)]()
Automated COVID-19 Lung CT Infection Segmentation using Frequency Domain Attention 

## Prerequisites

+ Linux
+ Python with numpy
+ NVIDIA GPU + CUDA 8.0 + CuDNNv5.1
+ pytorch 4.0/4.1
+ torchvision

## Getting started

- Clone this repo 
- Install requirements



**+ Train the model:**

    python train.py 

**+ Test the model:**

    python test.py
